# SQL-Portfolio-Project

# 📊 SQL Portfolio Project – Superstore Sales Analysis

## Overview

This repository presents a comprehensive SQL-based analysis of the **Superstore Sales** dataset, demonstrating the practical application of SQL for business intelligence and data analytics. The project focuses on extracting actionable insights from transactional sales data to support data-driven decision-making.

Using MySQL, I performed data exploration, aggregation, ranking, and trend analysis to evaluate sales performance, customer behavior, regional profitability, and product performance. Additionally, an interactive Tableau dashboard has been developed to visualize the analytical findings.



## Objectives

The primary objectives of this project are to:

* Analyze sales performance across different regions and product categories.
* Identify top-performing products and customers.
* Evaluate customer lifetime value.
* Monitor monthly revenue trends and growth.
* Demonstrate proficiency in SQL for solving real-world business problems.

## Business Problems Solved

### 1. Revenue Analysis by Product Sub-Category

* Calculated total sales generated by each product sub-category.
* Ranked sub-categories based on revenue.
* Evaluated profit margins to identify the most profitable products.

### 2. Regional Performance Analysis

* Compared sales performance across different regions.
* Calculated average order value.
* Identified regions with the highest customer engagement and revenue.

### 3. Customer Lifetime Value Analysis

* Measured total revenue generated by each customer.
* Ranked customers according to lifetime spending.
* Identified each customer's latest purchase date.

### 4. Monthly Revenue Trend Analysis

* Aggregated monthly sales performance.
* Calculated month-over-month revenue growth using window functions.
* Identified seasonal sales trends.

### 5. Category Performance Ranking

* Ranked product sub-categories within each category.
* Determined the highest revenue-generating products using analytical functions.
